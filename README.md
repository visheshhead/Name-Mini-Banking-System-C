# Mini Banking System in C

A console-based Mini Banking System built in C language that simulates basic banking operations and demonstrates file handling, structured data management, and menu-driven applications.
## Features
- Create a new bank account and save details in `.txt` file
- Deposit money
- Withdraw money
- View account details and balance
- Fully menu-driven console application
- File Handling for persistent storage
